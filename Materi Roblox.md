## Mini Adventure Game

Mini Game Adventure berisi **Penjelajahan dan Mencari Item** 

Buka Roblox Studio dan pilih template flat terrain.

- Terrain
    
    Dalam membuat sebuah lingkungan dalam game, kita memerlukan sebuah Tools yang bernama **Terrain Editor.**
    
    ![image.png](image%201.png)
    
    [https://filemanager.tmdsite.my.id/api/files/presigned/a13b0814-367b-4daf-86ac-bb14fdb8f290](https://filemanager.tmdsite.my.id/api/files/presigned/a13b0814-367b-4daf-86ac-bb14fdb8f290)
    
    1. Buka Roblox Studio > Templates > Flat Terrain
    2. Pada bagian tab Home, klik icon Terrain. Ini akan membuka Terrain Editor pada sisi kiri layar-mu
        
        [https://filemanager.tmdsite.my.id/api/files/presigned/a13b07c1-59c3-4833-8ba0-4df14e104ed2](https://filemanager.tmdsite.my.id/api/files/presigned/a13b07c1-59c3-4833-8ba0-4df14e104ed2)
        
    3. Pada **Terrain Editor,** pilih tab **Edit**. lalu gunakan **Paint Tool** untuk mengubah material sesuai keinginanmu. Tools paint dapat digunakan untuk **menggambar dengan material** lain di Terrain kamu. Kamu bisa mengubahnya menjadi Air, Batu, bahkan Lava.
        
        [https://filemanager.tmdsite.my.id/api/files/presigned/a13b089f-3216-419c-a6f0-50f38c67abf2](https://filemanager.tmdsite.my.id/api/files/presigned/a13b089f-3216-419c-a6f0-50f38c67abf2)
        
    4. Untuk mengubah material, pilih **Material Setting** -> **Material yang kamu inginkan. Material Setting** berisi pilihan material yang dapat mengubah dasar material (Source Material) menjadi material target (Target Material).
        
        [https://filemanager.tmdsite.my.id/api/files/presigned/a13b0ab9-1d70-4c5f-a974-a9e1b510b074](https://filemanager.tmdsite.my.id/api/files/presigned/a13b0ab9-1d70-4c5f-a974-a9e1b510b074)
        
    5. Gambarlah desain lingkungan terrain sesuai dengan keinginanmu.
        
        ![](https://academy.tmdcdn.my.id/uploads/photos/shares/Roblox/Roblox%20Coder/Meet%204/1.5.png)
        
    6. Buatlah bukit menggunakan tool **Add dan Subtract**.  Sesuaikan dengan rencana desainmu**.**
        
        ![](https://academy.tmdcdn.my.id/uploads/photos/shares/Roblox/Roblox%20Coder/Meet%204/add%20subtract%20met4.png)
        
    
- **Harvestable Item**
    
    Tujuan akhir dari mini game adventure ini adalah mengumpulkan **Harvestable Items**, yaitu Cupcake.
    
    ![image.png](image%202.png)
    
    1. Pada **Explore Tab** -> **Workspace** -> **Klik Kanan** -> **Model -**> **Klik kanan** -> **Rename** menjadi Cupcake.
        
        [https://filemanager.tmdsite.my.id/api/files/presigned/a150b267-f100-427f-879a-3694775ab5c5](https://filemanager.tmdsite.my.id/api/files/presigned/a150b267-f100-427f-879a-3694775ab5c5)
        
    
    ![image.png](image%203.png)
    
    Explore Tab biasanya ada di bagian kanan layar kerjamu. Namun jika tidak muncul kamu bisa membuka dibagian tab **Home -> pilih Explore Tab.**
    
    [https://filemanager.tmdsite.my.id/api/files/presigned/a13b0d10-fcd4-4aa3-aea3-5b729f6fa0cc](https://filemanager.tmdsite.my.id/api/files/presigned/a13b0d10-fcd4-4aa3-aea3-5b729f6fa0cc)
    
    List hirarki pada Explore Tab lebih sering disebut dengan **Parent-Child Relationship.**
    
    ![image.png](image%204.png)
    
    **Untuk membuat sebuah object menjadi Child dari sebuah object cukup dengan drag-and-drop ke dalam object parent yang diinginkan.**
    
    Dalam Explore Tab sudah terdapat Object - Object yang disebut dengan Default Object. Object ini adalah Object yang penting dan harus ada di Engine Game Roblox. Default Object dibagi kedalam 4 bagian besar, yaitu :
    
    ![](https://academy.tmdcdn.my.id/uploads/photos/shares/Roblox/Roblox%20Explorer/Meeting%204/objectiv.png)
    
    1. Tambahkan beberapa **Part** lalu **susunanlah menjadi Cupcake** seperti ini.
        
        ![](https://academy.tmdcdn.my.id/uploads/photos/shares/Roblox/Roblox%20Explorer/Meeting%204/ck.png)
        
    2. **Select semua Part** dan pilih **Union**. Ini akan membuat semua part dalam model tersebut menjadi satu.
        
        ![](https://academy.tmdcdn.my.id/uploads/photos/shares/Roblox/Roblox%20Coder/Meet%204/Part%202%20Modelling%20Harvestable%20Item%20(Harvestable%20Item).gif)
        
    3. Pastikan part telah di **Anchor** agar terkunci di posisinya. Jika tidak maka part akan jatuh atau menggelinding
        
        [https://filemanager.tmdsite.my.id/api/files/presigned/a13b0e12-532e-4b2d-a8b0-c1461397988e](https://filemanager.tmdsite.my.id/api/files/presigned/a13b0e12-532e-4b2d-a8b0-c1461397988e)
        
    4. Lalu **masukkan** ke dalam sebuah **Model dengan nama Cupcake**.
        
        [https://filemanager.tmdsite.my.id/api/files/presigned/a150d225-008e-4b23-aebd-1ff612bfa028](https://filemanager.tmdsite.my.id/api/files/presigned/a150d225-008e-4b23-aebd-1ff612bfa028)
        
    5. Dalam Model Cupcake tadi tambahkan objek **BoolValue.** Ganti **nama BoolValue** tersebut menjadi **canHarvest**. Aktifkan Valuenya dengan cara klik **Properties -> Value -> Centang.**
        
        ![](https://academy.tmdcdn.my.id/uploads/photos/shares/Roblox/Roblox%20Coder/Meet%204/Part%206%20Modelling%20Harvestable%20Item%20(Harvestable%20Item).gif)
        
- Spoon Tool
    
    Untuk membuat tool langsung berada ada pada player pada saat pertama kali start. Tool akan ditambahkan di folder **Starterpack**.
    
    1. Download terlebih dahulu starter tool berikut:
    Tombol yang terhubung ke(https://bit.ly/SpoonRoblox) 
    2. Pada **Tab Explore**, dibagian bawah Workspace, **klik kanan** pada **StarterPack**. Lalu **Insert > Import Roblox Model > Pilih Starter Tool yang sudah kamu download tadi.**
        
        [https://filemanager.tmdsite.my.id/api/files/presigned/a140946e-7f20-4847-b771-915db2261d47](https://filemanager.tmdsite.my.id/api/files/presigned/a140946e-7f20-4847-b771-915db2261d47)
        
    3. **Rename Tool** yang sudah ditambahkan, dengan nama sesuai yang ingin kamu tampilkan pada game. Contoh : **Scoop**.
    
        
        [https://filemanager.tmdsite.my.id/api/files/presigned/a14095bc-a577-4e31-993f-db96653a8a5c](https://filemanager.tmdsite.my.id/api/files/presigned/a14095bc-a577-4e31-993f-db96653a8a5c)
        
- Coding For Spoon
    
    Kita akan membuat kode, **Jika Tool menyentuh Harvestable Item dan player masih memiliki spaces** untuk itemnya, maka **Item akan bertambah 1** pada Leaderboard.
    
    Lalu **Harvestable Item akan menghilang beberapa detik** sebelum akhirnya nanti muncul kembali.
    
    Yuk ikuti langkah berikut !
    
    1. Pada **Tool**, tambahkan sebuah **script** didalamnya seperti gambar di bawah ini. Lalu **Rename script** tersebut menjadi **ToolScript**.
        
        ![](https://tmd-academy-dev.s3.ap-southeast-1.amazonaws.com/uploads/photos/shares/Roblox/ROBLOX2/scrip.png)
        
    2. Seperti sebelumnya hapus kode yang ada pada Script lalu tambahkan kode berikut.
        
        ![image.png](image%205.png)
        
    
    Kode diatas adalah kode untuk membuat variable bernama tool dan scoop. `local **scoop = tool.Scoop**` **berfungsi untuk menyimpan part Scoop ke dalam variable.**
    
    Sekarang kita akan membuat kondisi dimana **ketika Scoop menyentuh Cupcake akan mengubah nilai dari Boolvalue** dan mencetak “**Found an Item**“.
    
    1. Untuk membuatnya kita perlu membuat sebuah function bernama `onTouch()` dengan parameter `partTouch`.
        
        ![image.png](image%206.png)
        
    2. **Tambahkan kode untuk membuat kode print** didalam conditional statement untuk testing apakah code yang dibuat berjalan dengan baik. Berikut adalah codenya:

    1. Tambahkan kode berikut **pada ToolScript** setelah kode `local **scoop = tool.Scoop**`

    Variable-variable ini berfungsi untuk **membuat leadeboard dapat menghitung berapa jumlah Gold dan Item.**
