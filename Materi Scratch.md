# Apa Itu Scratch?

    ### CHALLANGE
    
    Tambahkan coding agar **Player mendapat skor ketika laser berhasil mengenai alien.** 
    
    Kamu hanya perlu  menambahkan **Variable Score** dan letakan pada susunan **code alien** seperti gambar di samping
    
    ![](https://academy.tmdcdn.my.id/uploads/photos/shares/Kids/Kids%20Self%20Learning%20Xplorer/Meeting%2021/img/28.png)
    
    Tambahkan **Sprite Win** dengan menggunakan **Paint Sprite > Text.** Berikan Coding agar **jika player mendapat skor 10, semua sprite hide dan sprite ini muncul If score = 10 > Show**
    
    ![](https://academy.tmdcdn.my.id/uploads/photos/shares/Kids/Kids%20Self%20Learning%20Xplorer/Meeting%2021/img/29.png)
    
    Tambahkan **Sprite Lose** dengan menggunakan **Paint Sprite > Text.** Berikan Coding agar **jika player hancur, semua sprite hide dan sprite ini akan muncul When I Receive Hit > Show**
    
    Tambahkan **Tombol Start Ketika game dimulai semua sprite hide dan hanya menampilkan tombol start.Ketika start di klik semua sprite akan muncul dan game dimulai.**
